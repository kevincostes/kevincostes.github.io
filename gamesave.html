GameSave

This project is about the development process of a web platform where the user can buy servers for video games that are deployed automatically, integrating our own cryptocurrency, and where the user pays only for the resources they need.

Github repository: Click here!

Requirements:

  • Python (version 3.10)
  • Django (version 4.2)
  • Other dependencies (see requirements.txt)
  • You need to have Docker installed on your computer (Docker Desktop).
  • MetaMask extension installed in your browser.
